OpenAI GPTs cli client written in haxe

# Building
To build gptcli, you can simply run
```
haxe build.hxml
```

# Requisites

- [Haxe](https://haxe.org)

# Intended use

- quickly querying OpenAI's GPTs from cli without using heavy web interface.
- run `gptcli --help` for more info
